# Upload & Content Types

**Supported formats:** PDFs, web articles, text, CSV, RSS feeds  
**Coming soon:** Images, audio, email ingestion

### Supported Content Types

**Documents:**
- PDF files (with OCR for scanned documents)
- Web articles (via URL)
- Text files (.txt, .md)
- CSV files (each row becomes a separate asset)

**Processing:**
- Extract text content from each page
- Preserve document structure and metadata
- Handle scanned PDFs with OCR capabilities
- Create child assets for individual pages

**Use Cases:** Research papers, government documents, reports, legislation, news monitoring, blog analysis, survey data, financial records
