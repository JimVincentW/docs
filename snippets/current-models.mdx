# Current Models

**Cloud Models** (require API keys):
- **OpenAI GPT-4** — Best for complex analysis
- **OpenAI GPT-3.5** — Faster, more cost effective
- **Google Gemini** — Good alternative to OpenAI

**Local Models** (via Ollama):
- **Llama 2/3** — Free, privacy focused
- **Mistral** — Fast and efficient
- **Code Llama** — Good for structured data extraction

Configure API keys in the web interface or run Ollama locally for complete privacy.
