### Self-Hosted with Docker
1. Clone the repository and prepare the environment:
```bash
git clone https://github.com/open-politics/open-politics-hq.git
cd open-politics-hq
bash prepare.sh
cp .env.example .env
```

Log in with the FIRST_SUPERUSER and FIRST_SUPERUSER_PASSWORD set in the .env file.
